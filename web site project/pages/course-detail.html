Course Overview

This comprehensive bootcamp will take you from absolute beginner to a confident web developer. You will learn HTML5, CSS3, JavaScript, Responsive Design, and real project development.

What You'll Learn

  1. Build responsive websites
  2. Master HTML, CSS, and JavaScript
  3. Use Git and GitHub
  4. Work with real projects and tasks

Requirements

  • No prior experience required
  • A computer with internet access
  • Motivation and commitment

Course Level: Beginner

Duration: 8 Weeks

Time Commitment: 5–7 hours per week

Course Syllabus

Module 1: Introduction to HTML

Lesson 1: HTML Basics

Understanding HTML structure and tags.

Lesson 2: Lists and Links

Create ordered/unordered lists and hyperlinks.

Lesson 3: Images and Media

Working with images, audio, and video.

Module 2: CSS Styling

Lesson 1: CSS Syntax

Selectors, properties, and values.

Lesson 2: Flexbox & Grid

Layout techniques for modern websites.

Lesson 3: Responsive Design

Media queries and adaptive layouts.

Module 3: JavaScript Foundations

Lesson 1: Variables & Data Types

How JavaScript stores and handles data.

Lesson 2: Functions

Reusable blocks of code.

Lesson 3: DOM Manipulation

Create interactive web pages.
